## 目的
- MVP重构IDouban
- Lambda使用

## Android支持包新增控件
- RecyclerView
- CardView
- CoordinatorLayout
- AppBarLayout
- Toolbar

## 网络请求框架
- Retrofit

## 图片异步加载
- Picasso


网络数据来源于豆瓣的公开接口： [豆瓣电影V2](https://developers.douban.com/wiki/?title=movie_v2)
